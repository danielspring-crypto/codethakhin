Install Python For Computer
  1. www.python.org ကိုသွားပါ။
  2. Downloads button ကိုနှိပ်ပြီး (Windows x86-64 executable installer) ကို Download ဆွဲပါ။
  3. ထို့နောက် မိမိစက်ထဲရှိ Downloads folder ထဲမှ python3.9.exe file အား install ပြုလုပ်ပါ။
  4. Install ပြုလုပ်ရာတွင် add python3.9 to path ဆိုသည့်နေရာတွင် အမှန်ခြစ်ပေးပါ။
  5. Install လုပ်ပြီးသွားသောအခါ command line တွင် python ဟုရိုက်ပါ။ python console ကျလာပါက installation ပြီးပြီဖြစ်ပါသည်။
Python File တွေကို .py ဆိုတဲ့ extension နဲ့ save ပါတယ်။ Python Code တွေရေးသားရာမှာ မိမိ အဆင်ပြေတဲ့ text editor / IDE ကိုရွေးချယ်အသုံးပြုနိုင်ပါတယ်။

အသုံးများဆုံး software တွေကတော့
  1. VSCode (recommended)
  2. Sublime Text
  3. PyCharm တို့ဖြစ်ပါတယ်။
