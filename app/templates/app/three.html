Video Tutorial ကို January 1ရက်နေ့တင်ပေးပါမည်။

Syntax ဆိုတာဟာ Program တစ်ခုတည်ဆောက်ရာမှာအသုံးပြုတဲ့ code ရေးသားနည်းဖြစ်ပါတယ်။ Syntax မှားရင် program run တဲ့အခါ error တက်နိုင်တာမို့လို့ syntax ကိုအထူးဂရုစိုက်သင့်ပါတယ်။ သို့ပေမယ့် Python Programming ရဲ့ ရိုးရှင်းလွယ်ကူတဲ့ syntax တွေကြောင့် pythoner တို့အတွက် error နည်းပြီး လေ့လာရလွယ်ကူစေမှာပါ။
ပထမဆုံး program တစ်ခုကို စတင်ရေးကြည့်ကြပါစို့။ VSCode မှာ newfile တစ်ခုပြုလုပ်ပြီး .py ဖြင့် save ပါ။ အောက်ပါ code ကိုရေးပါ။ file ကို save ပြီး run ကြည့်လိုက်တဲ့အခါမှာ Hello World ဆိုတဲ့ Output ကိုမြင်ရပါလိမ့်မယ်။

filename : basic_syntax.py print("Hello World")
Explanation
  • print ဆိုတဲ့ syntax လေးက function name ဖြစ်ပြီး () လေးက function အမှတ်အသားဖြစ်ပါတယ်
  • ဒါ့ကြောင့် print() ဟာ function တစ်ခုဖြစ်ပြီး Program Result ထုတ်တဲ့ အခါမှာသုံးပါတယ်
  • အထဲက Hello World ကတော့ output ပြချင်တဲ့ စာသားဖြစ်ပြီး character ဖြစ်တဲ့အတွက် " " ထဲထည့်ရေးရပါတယ်
  • Programming Terms အရ character တွေကို string (str = အတိုကောက်) ဟုခေါ်ပါတယ်။
Error မဖြစ်ဖို့သတိထားရမှာတွေကတော့
  • Syntax Error မဖြစ်ဖို့ စာသားတွေမှန်ဖို့လိုပါတယ်။ capital letter small letter လွဲလို့မရပါဘူး။
  • Identataion Error မဖြစ်ဖို့ syntax အစနေရာတွေမှာ space တွေ tab တွေမပါဖို့ ဂရုစိုက်ပေးရပါမယ်။
