For

" for " ကို သတ်မှတ်ထားသောအကြိမ်အရေအတွက်အထိ ထပ်ခါတလဲလဲအလုပ်လုပ်လိုသည့်အခါသုံးပါတယ်။

filename : for_loop.py text = "Programming"
for x in text:
  print(x)
Explanation
  • Variable Name - text ထဲကို "Programming" လို့ assign လုပ်ထားပါတယ်။
  • for loop သုံးပြီး text ထဲက data တွေကို တစ်ကြိမ်မှာတစ်ခု output ပြပါမယ်။
  • Output Result ကို x ဆိုတဲ့ variable ထဲမှာ တစ်ကြိမ်မှာတစ်ခု ပြပါတယ်။
While

"while" ကို loop အပြင် condition နဲ့ပါ တွဲပြီးအသုံးပြုတဲ့ program တွေမှာသုံးပါတယ်။

filename : while_loop.py i = 1
while i < 6:
  print(i)
  i += 1

Explanation
  • Variable Name - i ထဲကို 1 assign လုပ်ထားပါတယ်။
  • while loop ကိုသုံးပြီး i သည် 6 ထက်ငယ်တဲ့အခါမှာ အလုပ်လုပ်စေတဲ့ condition စစ်ထားပါတယ်။
  • i ရလဒ် 6 ထက်မကြီးမချင်း 1 ကိုထပ်တလဲလဲပေါင်းပေးကာ result ထုတ်ပေးသွားပါတယ်။
