Function ဆိုတာ program တစ်ခုလုံးရဲ့ အစိတ်အပိုင်းများ သီးခြားအလုပ်လုပ်ဖို့ ရေးထားတဲ့ code ဖြစ်ပါတယ်။၊
ဥပမာ - smart ဖုန်းတစ်လုံးဟာ program ကြီးတစ်ခုဆိုရင် camera ရိုက်တာ phone ခေါ်တာ သီချင်းနားထောင်တာ တွေဟာ function တွေ ဖြစ်ပါတယ်။
Simple Function တစ်ခုကိုစတင်ရေးကြည့်ပါမယ်။ အောက်ပါအတိုင်းရေးပြီး run လိုက်ရင်ဘာမှပေါ်လာမှာမဟုတ်ပါဘူး။ ဘာလို့လဲဆိုတော့ function တစ်ခုဖန်တီးပြီးတာနဲ့ function name နဲ့ call ခေါ်မှ အလုပ်လုပ်တာဖြစ်ပါတယ်။ ဒါ့ကြောင့် greet() လို့ အောက်မှာ function call ခေါ်ပေးရပါမယ်။

filename : function.py
def greet():
  print("Mingalarbar")
greet()
parameter, argument နဲ့ object တွေအကြောင်းကို ဆက်လက်လေ့လာကြပါမယ်။
Parameter ဟာ function တည်ဆောက်ရာမှာ data store ဖို့ variable တစ်ခုအနေနဲ့ အလုပ်လုပ်ပါတယ်။
Argument ဟာ function ထဲကိုထည့်မယ့်တန်ဖိုးဖြစ်ပါတယ်။
Function Call ခေါ်တဲ့အခါမှာ assign လုပ်လို့ရတဲ့ variable ကို object လို့ခေါ်ပါတယ်။

filename : function2.py def greet(username):
  print("Hello ", username)
person1 = greet("Daniel")
Explanation
  • greet() ဆိုတဲ့ နှုတ်ဆက်တဲ့ function တစ်ခုပြုလုပ်ထားပါတယ်။
  • greet က function name ဖြစ်ပြီး၊ parameter ကတော့ username ဖြစ်ပါတယ်။
  • Function အလုပ်လုပ်ရန်အတွက် Function call ခေါ်ရပါတယ်။
  • function call ထဲမှာရှိတဲ့ Daniel က argument ဖြစ်ပါတယ်
  • person1 ဟာ variable ဖြစ်ပေမယ့်၊ function assign လုပ်လိုက်တဲ့အခါ object လို့ခေါ်ပါတယ်။
Types of Arguments
  1. Positional Argument (အသုံးများ)
  2. Keyword Argument
  3. Arbitary Argument
Positional Argument

parameter 2 ခု ထက်ပိုရှိတဲ့အခါမှာ သုံးပါတယ်။ အရင်လာတဲ့ argument တန်ဖိုးကို function ရဲ့ ပထမနေရာမှာထည့်တာမျိုးပါ။

filename : positional_argument.py def greet(username, age):
  print("Hello, I am ", username)
  print("I am ", age)
person1 = greet("Daniel Dufrank", 22)
Keyword Argument

parameter 2 ခု ထက်ပိုရှိတဲ့အခါမှာ သုံးပါတယ်။ နေရာတကျဖြစ်စရာမလိုပဲ pythoner တို့တွဲချင်တဲ့ parameter နဲ့ argument နဲ့ တွဲလို့ရတာကိုဆိုလိုပါတယ်။

filename : keyword_argument.py def greet(username, age):
  print("Hello, I am ", username)
  print("I am ", age)
person1 = greet(username = "Daniel", age = 22)
Arbitary Argument

Argumet ဘယ်နှခုမှန်း parameter ကမသိရတဲ့အခါသုံးပါတယ်။

filename : arbitary_argument.py def profile(*info):
  print(info)
person1 = profile("Daniel", 22, "Male")
