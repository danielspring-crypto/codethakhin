Python Programming Basic

Tutorial Video

Video Tutorial ကို January 1ရက်နေ့တင်ပေးပါမည်။

Variable ဆိုတဲ့သဘောတရားဟာ ငယ်ငယ်တုန်းက မသိကိန်းပုစာအတွက်အချက်မှာ တန်ဖိုးထည့်ရသလိုမျိုးပါပဲ။ မောင်မောင် ရဲ့အသက်ကို x ဟုထားပါ။ မောင်မောင်သည် ကျော်ကျော်ထက် 2 နှစ်ပိုကြီးသည်လို့ဆိုလိုတာနဲ့ x = 2y လို့ ရေးရသလိုပါပဲ။

variable တစ်ခုကို data သွင်းပြီး output ထုတ်တာကို စမ်းကြည့်ကြပါမယ်။

filename : variable.py x = "I love Programming"
print(x)

Explanation
  • x သည် variable ဖြစ်ပြီး "I love programming" ဆိုတဲ့ data ကိုထည့်ထားပါတယ်။
  • ထို့နောက် x ထဲမှာရှိတဲ့တန်ဖိုးကို print() နဲ့ထုတ်ပြီးပြတဲ့အခါ x ရဲ့ data တွေထွက်လာပါတယ်။
Variable Name ပေးတဲ့အခါမှာ pythoner တို့ အနေနဲ့ မိမိရေးတဲ့ program အပေါ်မူတည်ပြီးဆီလျော်အောင်ပေးဖို့လိုအပ်ပါတယ်။
Variable Name ပေးတဲ့အခါမှာ အောက်ပါပုံစံမျိုး တွေမပါဝင်အောင်အထူး သတိထားရပါမယ်။
  • special character (eg: !@#$%^&*)
  • ရှေ့ဆုံးစကားလုံး ကိန်းဂဏန်း (eg: 1x)
  • space and tab (eg: 1 x)
  • programming syntax (eg: type, class, for)
Variable ထဲမှာထည့်သွင်းနိုင်တဲ့ datatype အမျိုးမျိုးရှိပြီး builtin function ဖြစ်တဲ့ type() ကိုသုံး၍ စစ်ကြည့်နိုင်ပါတယ်။

filename : check_variable.py x = "I love Programming"
print(type(x))

Explanation
  • Output မှာ str လို့ပြတဲ့အတွက် x ထဲမှာရှိတဲ့ datatype သည် string အမျိုးအစားဖြစ်ပါတယ်။

String Concatenation
String Concatenation ဆိုတာဟာ data နှစ်ခုနဲ့ နှစ်ခုထက်ပိုတဲ့ output တွေပြဖို့ရာအတွက်သုံးပါတယ်။ concatenate လုပ်တဲ့အခါ ',' ခံပြီးသုံးရပါမယ်။

filename : string_concatenate.py print("I love ", "Programming")
Variable နဲ့ လဲတွဲထုတ်လို့ရပါတယ်။

filename : concatenate.py x = "Programming"
print("I love ", x)
