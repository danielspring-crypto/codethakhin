Action 3 ခုထက်ပိုပြီးအလုပ်လုပ်ရမယ့် calculator program တစ်ခုကိုရေးကြည့်ပါမယ်။ User Input ဂဏန်း 2 ခုရိုက်ထည့်ပေးရပြီး ( +, -, *, / ) ရွေးချယ်ကာ တွက်ချက်အဖြေထုတ်ပေးတဲ့ simple calculator program ဖြစ်ပါတယ်။

filename : simple_calculator.py num1 = int(input("Enter Number 1 : "))
num2 = int(input("Enter Number 2 : "))
print(" +, -, *, / ")
choice = input("Enter Your Choice : ")
if choice == "+":
  answer = num1 + num2
  print(answer)
elif choice == "-":
  answer = num1 - num2
  print(answer)
elif choice == "*":
  answer = num1 * num2
  print(answer)
elif choice == "/":
  answer = num1 / num2
  print(answer)
else:
  print("Try Again!")
Explanation
  • ပထမဆုံး user input နှစ်ခုစာ input တောင်းပါတယ်။
  • ပြီးတော့ User ရဲ့ choice ကိုတောင်းပါတယ်။
  • User ရဲ့ choice က + operator နဲ့ညီရင် ပေါင်းတဲ့အလုပ်လုပ်ပါမယ်။
  • Programming မှာ ညီတယ်ဆိုတဲ့ သဘောတရားကို " == " operator နဲ့ပြပါတယ်။
  • " = " ဆိုတဲ့ assignment operator ကိုတော့ variable ထဲမှာ value ထည့်ဖို့အတွက်သုံးပါတယ်။
