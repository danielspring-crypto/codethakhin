input( )
user ရဲ့ input တောင်းတဲ့အခါသုံးပါတယ်။ အောက်ဖော်ပြပါ input.py file ကို Run ကြည့်လိုက်တဲ့အခါ program က Enter Something ဆိုပြီး input တောင်းတာကိုတွေ့ရပါမယ်။ Input တစ်ခုကိုရိုက်ထည့်ပြီး enter ခေါက်လိုက်တဲ့အခါ pythoner တို့ရိုက်ထည့်ထားတဲ့ input ကို result ထုတ်ပြထားတာတွေ့ရပါမယ်။

filename : input.py x = input("Enter Something : ")
print("You entered ", x)

input( ) function မှာ default datatype အနေနဲ့ user ထဲ့လိုက်တဲ့ data တိုင်းကို str (string method) အနေနဲ့မှတ်ယူပါတယ်။ type( ) နဲ့ လည်းစစ်ကြည့်လို့ရနိုင်ပါတယ်။

filename : input.py x = input("Enter Something : ")
print("You entered ", x)
print(type(x))
int( )
ကိန်းဂဏန်းပြောင်းချင်တဲ့အခါသုံးတဲ့ function ဖြစ်ပါတယ်။

filename : int.py x = int(input("Enter number : "))
print(x)
print(type(x))
str( )
စာသားအဖြစ်ပြောင်းချင်တဲ့အခါသုံးတဲ့ function ဖြစ်ပါတယ်။ value ဖြစ်တဲ့ 12345 ဆိုတဲ့ int (integer) ကို str (string) အဖြစ်ပြောင်းပြထားတာကိုတွေ့ရပါမယ်။

filename : str.py x = 12345
print(type(x))
x = str(x)
print(type(x))
len( )
စာလုံးအရေအတွက် စစ်ချငတဲ့နေရာမှာသုံးပါတယ်။

filename : len.py x = "Hello Programming"
print(len(x))
