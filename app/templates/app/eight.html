Data Structure ဆိုတာ data တွေ သိမ်းဆည်းဖို့ ပြင်ဆင်ဖို့ စနစ်တကျစီမံခန့်ခွဲဖို့ အတွက်ဖြစ်ပါတယ်။ Computer ရဲ့ memory ထဲမှာ data တွေ store သလိုမျိုး program ထဲမှာ data တွေ သိမ်းဆည်းတာဖြစ်ပါတယ်။
Python မှာ သုံးတဲ့ DataStructure အမျိုးအစားတွေကတော့

  2. Tuple

List

List ထဲမှာ data တွေ store လုပ်ပြီး result ထုတ်ကြည့်ပါမယ်။ Python သည် Dynamic Type ဖြစ်သောကြောင့် list ထဲမှာ မည်သည့် datatype ကိုမဆိုထည့်သွင်းနိုင်ပါတယ်။

x = [1, 2, "programming", "Networking"]
print(x)
List Indexing

သင်္ချာရဲ့ ကိန်းဂဏန်းရေတွက်ခြင်းမှာ အပြည့်ကိန်း နဲ့ သဘာဝကိန်းဆိုပြီး ရှိပါတယ်။ အပြည့်ကိန်းကို ၀ ကနေစတင် ရေတွက်ပြီး သဘာဝကိန်းကို ၁ ကစရေတွက်ပါတယ်။ list သည် အပြည့်ကိန်းသဘောတရားကိုလိုက်နာတဲ့အတွက် ၀ ကနေစတင်ရေတွက်ပါတယ်။

filename : list_indexing.py x = [1, 2, 'hello']
print(x[0])
print(x[1])
print(x[2])
Explanation
  • Variable x ထဲမှာ list value တွေထည့်သွင်းထားပါတယ်။
  • print(x[0]) ကိုထုတ်လိုက်တဲ့အခါ အပြည့်ကိန်းသဘောတရားအရ ရေတွက်သောကြောင့် 1 လို့ရလာမှာဖြစ်ပါတယ်။
List Update

ဆက်လက်ပြီး List မှာ data တွေပြင်ဆင်တာနဲ့ ပေါင်းထည့်တာကို ဆက်လက်လေ့လာကြည့်ပါမယ်

filename : list_update.py x = [1, 2, 3, 4, 5]
x[0] = 100
print(x)

Explanation
  • Variable x ထဲမှာ list value တွေထည့်သွင်းထားပါတယ်။
  • List ရဲ့ပထမ နေရာကို 100 နဲ့ update လုပ်ပါတယ်။
  • print(x) ထုတ်တဲ့အခါမှာ 1 မဟုတ်ပဲ 100 ပြောင်းသွားတာတွေ့ရပါမယ်။
List Append

Data တွေ append လုပ်တဲ့အခါ default အနေနဲ့ list ရဲ့နောက်ဆုံးကနေထည့်ပါတယ်။

filename : list_append.py x = [1, 2, 3, 4, 5]
x.apppend(6)
print(x)

Explanation
  • Variable x ထဲမှာ list value တွေထည့်သွင်းထားပါတယ်။
  • List ထဲကို append() function နဲ့ 6 ပေါင်းထည့်ပါတယ်။
  • print(x) ထုတ်တဲ့အခါမှာ 1,2,3,4,5,6 ဆိုပြီးရပါတယ်။
Tuple နဲ့ Set ဟာလည်း datastore လုပ်ဖို့သုံးပါတယ်။ သို့သော် Tuple နဲ့ Set ဟာ List လို update / delete လုပ်လို့မရပါဘူး။
