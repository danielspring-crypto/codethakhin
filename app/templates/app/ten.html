Class ဆိုတာ Function တွေ Data တွေကို တစ်စုတစ်ဝေးထဲ ထားထားတဲ့အစုအဝေးတစ်ခုပါပဲ။ Class သုံးခြင်းအားဖြင့် program တွေရေးတဲ့အခါမှာ ပိုပြီးအထောက်အကူဖြစ်စေပါတယ်။ class ကိုသေချာနားလည်ပြီး အသုံးချနိုင်ပြီဆိုတာနဲ့ advanced အပိုင်းတွေဖြစ်တဲ့ web development / android app / desktop application နဲ့ AI ပိုင်းတွေကိုပါ လေ့လာနိုင်သွားမှာဖြစ်ပါတယ်။
Function တွေစုပြီး Simple Class လေးတစ်ခုစရေးကြည့်ပါမယ်။ Dog ဆိုတဲ့ class တစ်ခုတည်ဆောက်ပြီး အထဲမှာခွေးတစ်ကောင်ရဲ့ function အချို့ကို ရေးပြီး object တစ်ခုကနေသုံးကြည့်ပါမယ်။

filename : simple_class_method.py
class Dog():
  def sit(self):
    print("Dog is sitting ")
  def bark(self):
    print("Dog is Barking ")
dog1 = Dog()
dog1.sit()
dog1.bark()
Explanation
  • class keyword နဲ့ Dog ဆိုတဲ့ Class ကိုရေးပါတယ်။ Class Name ကို capital letter (example : Dog) နဲ့ရေးပါတယ်။ function name တွေကို small letter (example : greet) နဲ့ရေးပါတယ်။ Function နဲ့ Class ကို capital & small letter ကိုကြည့်ပြီးခွဲပါတယ်။
  • class တွေမှာ ရေးတဲ့ function တွေကို class ရဲ့ method လို့ခေါ်ပါတယ်။ function တွေကိုသာမန်အတိုင်းရေးလို့မရပဲ self ဆိုတဲ့ keyword သုံးပြီး class နဲ့ချိတ်ဆက်ရပါတယ်။
  • function တွေတုန်းကအတိုင်းပဲ class တစ်ခုတည်ဆောက်ပြီးတိုင်း object တစ်ခုပြုလုပ်ပြီး call ခေါ်အသုံးပြုရပါတယ်။
  • dog1 ဆိုတဲ့ object နဲ့ Dog class ကို call ခေါ်လိုက်ပါတယ်။ အခုဆိုရင် dog1 ဆိုတဲ့ object ဟာ Dog class ထဲမှာရှိတဲ့ sit() / bark() ဆိုတဲ့ function / method တွေကို ခေါ်သုံးနိုင်ပြီဖြစ်ပါတယ်။
ဆက်လက်ပြီး Dog ဆိုတဲ့ class ထဲမှာ data တွေထပ်ဖြည့်ပါမယ်။

filename : simple_class_data.py
class Dog():
  def __init__(self, name, age):
    self.name = name
    self.age = age
  def sit(self):
    print("Dog is Sitting ")
  def bark(self):
    print("Dog is Barking ")
dog1 = Dog("Aung Net", 5)
print("My Dog name is ", dog1.name)
print("It is ", dog1.age ,"years old")
dog1.sit()
dog1.bark()
Explanation
  • name, age စတဲ့ data တွေကို constructor method ဖြစ်တဲ့ ( __init__ ) နဲ့ထည့်ပြီးဖော်ပြပါတယ်။ class ကနေ access လုပ်နိုင်အောင် self နဲ့ link ချိတ်ပါတယ်။
  • dog1 object နဲ့ Dog class ကိုခေါ်တဲ့အခါမှာ name, age data တွေကိုထည့်သွင်းနိုင်ပါပြီ။
  • dog1.name, dog1.age စတဲ့ data တွေကို object ကနေ ခေါ်သုံးနိုင်ပါပြီ။
Inheritance

အမွေဆက်ခံခြင်းလို့ အဓိပါယ်ရပါတယ်။ ကိုယ်တိုင် ရေးထားတဲ့ မူလ class ကို program လိုအပ်ချက် အတွက် upgrade လုပ်တဲ့အခါမျိုး၊ external library file တွေ module တွေ ဆီက code တွေကို upgrade လုပ်ချင်တဲ့အခါသုံးပါတယ်။ မူရင်း class ကို parent class လို့ခေါ်ပြီး upgrade လုပ်တဲ့ class ကို child class လို့ခေါ်ပါတယ်။
မူရင်း parent class Dog ကို child class တစ်ခုမှာ inheritance လုပ်ပါမယ်။

filename : inheritance.py
class Dog():
  def __init__(self, name, age):
    self.name = name
    self.age = age
  def sit(self):
    print("Dog is Sitting ")
  def bark(self):
    print("Dog is Barking ")
class Hound(Dog):
  def __init__(self, name, age):
    super().__init__(name, age)
  def play(self):
    print("Dog is Playing ")
  def hunting(self):
    print("Dog is Hunting ")
dog1 = Hound("Aungnet", 5)
print(dog1.name)
print(dog1.age)
dog1.sit()
dog1.bark()
dog1.play()
dog1.hunting()
Explanation
  • Hound ဆိုတဲ့ class တစ်ခုဆောက်ပြီး Dog class ကို inherit လုပ်ထားပါတယ်။ Dog class ရဲ့ method တွေကိုခေါ်သုံးနိုင်ပြီဖြစ်ပါတယ်။
  • Dog class ရဲ့ Data တွေပါခေါ်သုံးလို့ရအောင် super() function နဲ့ Hound class မှာ inherit လုပ်ထားပါတယ်။
  • dog1 object တစ်ခုဆောက်ပြီး Hound class ကို call ခေါ်လိုက်တဲ့အခါမှာ parent class Dog class ထဲက data တွေ method တွေကိုပါ အသုံးပြုနိုင်မှာဖြစ်ပါတယ်။
